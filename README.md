God of Decision - Solution Code
=================================

Solution code for the Android Basics in Kotlin: God of Decision app.


Introduction
------------
The God of Decision app contains various UI elements for calculating a tip,
teaching about app structure, user input, and view binding.


Pre-requisites
--------------
* Experience with Kotlin syntax.
* How to create and run a project in Android Studio.


Getting Started
---------------
1. Install Android Studio, if you don't already have it.
2. Download the sample.
3. Import the sample into Android Studio.
4. Build and run the sample.
